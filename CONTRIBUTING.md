# Contributing to Mouse IA

Obrigado por considerar contribuir com o Mouse IA.

Este projeto está em fase inicial de desenvolvimento e a participação da comunidade é bem-vinda. Antes de começar, por favor leia este guia para entender o fluxo esperado.

## Como contribuir

1. Faça um fork do repositório.
2. Crie uma branch para a sua mudança.
3. Implemente as alterações com foco em clareza e qualidade.
4. Teste o que for possível antes de enviar a contribuição.
5. Abra um pull request descrevendo o que foi alterado e por quê.

## Padrões esperados

- Escreva código limpo e bem estruturado.
- Mantenha a documentação atualizada quando alterar comportamento ou arquitetura.
- Prefira mudanças pequenas e bem focadas.
- Use mensagens de commit claras e descritivas.

## Estrutura do projeto

O projeto é organizado em:

- backend: APIs, serviços e lógica de negócio
- frontend: interface e experiência do usuário
- docs: documentação técnica e de produto
- scripts e infraestrutura: automações e suporte operacional

## Áreas de contribuição

Você pode contribuir com:

- melhorias de código
- correções de bugs
- documentação
- testes
- arquitetura e organização do projeto
- sugestões de funcionalidade

## Regras básicas

- Não envie alterações sem contexto ou sem explicação.
- Respeite a arquitetura e os padrões já adotados.
- Evite mudanças muito amplas sem discussão prévia.
- Se estiver implementando algo novo, descreva a proposta no pull request.

## Relato de problemas

Se encontrar um problema, abra uma issue descrevendo:

- o comportamento esperado
- o comportamento observado
- passos para reproduzir
- contexto técnico relevante

## Licença

Ao contribuir, você concorda que suas alterações serão licenciadas sob a mesma licença do projeto.
